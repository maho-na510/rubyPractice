# rubyPractice

## DB・テーブル確認手法

## コンテナ内に入ってmysqlにログインする場合

以下の手順でログインする

```
docker exec -ti db /bin/bash
mysql -u root -p 
=> password
```

## phpMyadminを使って表示する場合

以下の手順で表示する

```
http://localhost:9999/にアクセス
```
